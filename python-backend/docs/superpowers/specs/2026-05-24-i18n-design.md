# i18n 国际化（中英双语）设计文档

**日期：** 2026-05-24  
**范围：** 前端 UI + 后端 AI Prompt 双语支持  
**策略：** 全局语言开关，UI 与文章生成语言联动

---

## 目标

在不改变数据库结构、路由体系、SSE 流程的前提下，为项目添加中英文切换能力：
- 前端 UI 所有文字可在中文 / 英文之间即时切换
- 切换后新建文章自动使用对应语言的 AI Prompt，生成对应语言的文章内容

---

## 架构概览

```
AppHeader [ZH | EN] 切换
  │
  ├─ vue-i18n locale 切换 → UI 文字即时更新
  ├─ localStorage 持久化 → 刷新后保持
  │
  └─ API 请求自动附带 language: 'zh' | 'en'
       │
       ▼
后端 ArticleCreateRequest.language
  │
  └─ 存入 ArticleState.language
       │
       └─ ArticleAgentService 调用 PromptConstant.get(key, language)
            ├─ 'zh' → 现有中文 Prompt（不动）
            └─ 'en' → 新增 _EN 变体
```

**不动的部分：** 数据库表结构、SQLAlchemy models、routers、前端路由、SSE 流程、agent 编排逻辑

### AppHeader 语言切换

- 在 AppHeader 右侧加 `[ZH] [EN]` 两个 tab 按钮，当前语言高亮
- 点击逻辑：
  1. `locale.value = 'en'`（或 `'zh'`）
  2. `localStorage.setItem('lang', locale.value)`
- 初始化时从 `localStorage.getItem('lang') ?? 'zh'` 读取

### API 联动

`api/article.ts` 的 `createArticle()` 调用从 `i18n.global.locale.value` 取当前语言，
自动附加 `language` 字段到请求体，无需用户手动选择。

```python
class ArticleState:
    def __init__(self, task_id, topic, style, user_description,
                 enabled_image_methods, language='zh'):   # 新增 language 参数
        ...
        self.language: str = language
```

### PromptConstant 变更

**`app/constants/prompt.py`**

策略：现有中文常量**完全不动**，每个需要双语的常量新增 `_EN` 后缀变体，
加一个 `get()` 静态方法做语言分发，找不到英文版时回退到中文。

```python
@staticmethod
def get(name: str, language: str = 'zh') -> str:
    """按语言取 Prompt，英文版缺失时回退中文。"""
    if language == 'en':
        return getattr(PromptConstant, f"{name}_EN",
                       getattr(PromptConstant, name))
    return getattr(PromptConstant, name)
```

**需要新增英文变体的常量：**

| 常量名 | 用途 |
|--------|------|
| `AGENT1_TITLE_PROMPT_EN` | 标题方案生成 |
| `AGENT2_DESCRIPTION_SECTION_EN` | 用户补充描述段 |
| `AGENT2_OUTLINE_PROMPT_EN` | 大纲生成 |
| `AGENT3_CONTENT_PROMPT_EN` | 正文生成 |
| `AGENT4_IMAGE_REQUIREMENTS_PROMPT_EN` | 配图需求分析 |
| `AI_MODIFY_OUTLINE_PROMPT_EN` | AI 修改大纲 |
| `STYLE_INSTRUCTIONS_EN` | 各风格说明（dict） |

`get_style_instruction()` 方法也加 `language` 参数：
```python
@staticmethod
def get_style_instruction(style: str, language: str = 'zh') -> str:
    instructions = (PromptConstant.STYLE_INSTRUCTIONS_EN
                    if language == 'en'
                    else PromptConstant.STYLE_INSTRUCTIONS)
    return instructions.get(style, '')
```

### ArticleAgentService 变更

每个 agent 方法的 prompt 构造从直接引用常量改为调用 `get()`：

```python
# 修改前
prompt = PromptConstant.AGENT1_TITLE_PROMPT.replace(...)

# 修改后
prompt = PromptConstant.get("AGENT1_TITLE_PROMPT", state.language).replace(...)
```

`get_style_instruction()` 调用同步加 `state.language` 参数。

### ArticleService 变更

`start_article_generation()` 构造 `ArticleState` 时传入 `language`：
```python
state = ArticleState(..., language=request.language)
```
